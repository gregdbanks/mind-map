Node Operations Test

Instructions:

  1. Open the mind map app at http://localhost:3001
  2. Load the demo map
  3. Test each operation below and verify it works

1. Add Node

✓ Hover over any node - action buttons should appear

✓ Click the green "+" button - a new child node should be created

✓ Select a node and press Enter - a new child node should be created

2. Edit Node

✓ Hover and click the blue "✎" button - editor should appear

✓ Select a node and press F2 - editor should appear

✓ Type new text and press Enter - node text should update

✓ Press Escape while editing - changes should be cancelled

3. Delete Node

✓ Hover and click the red "×" button - confirm dialog should appear

✓ Click OK in confirm dialog - node and children should be deleted

✓ Select a node and press Delete key - node should be deleted

4. Visual Feedback

✓ Action buttons only visible on hover

✓ Action buttons hidden during drag

✓ Tooltip shows on hover (not during edit)

5. Undo/Redo

✓ Ctrl+Z undoes last action

✓ Ctrl+Y redoes undone action
